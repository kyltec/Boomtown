## Server

## NOTE: Please note that a database is required to run the server.

Commands must be run from the `server` directory:

### Installation

- Clone this repo

```bash
cd server directory
npm install
```

### Run

```bash
npm run start:dev
```

## Client

Commands must be run from the `client` directory:

### Installation

```bash
cd client directory
npm install
```

### Run

```bash
npm start
```
